* Notes

** Task Status Guide for AI Agents

*** When to change status:
    - TODO → IN-PROGRESS: Starting work on complex task
    - TODO → WAITING: Simple task finished, needs human review
    - TODO → DONE: Simple task obviously complete
    - IN-PROGRESS → WAITING: Complex task finished, needs review (most common)
    - IN-PROGRESS → DONE: Complex task obviously complete
    - WAITING → DONE: Human approves the work
    
*** What to add in notes:
    - Date and "Agent:" prefix
    - Approach taken
    - Files modified
    - Questions for human reviewers
    - Links to relevant code or ADRs
    
*** Properties to maintain:
    - :CREATED: When task was created
    - :STARTED: When work began
    - :COMPLETED: When moved to WAITING or DONE
    - :PRIORITY: High/Medium/Low
    - :ASSIGNEE: Who's working on it
